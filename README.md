# AI-Powered Job Platform + Autonomous Agents

This repository contains a fully automated job-search, CV creation,
auto-application, AI agents, marketing autoposters, applicant tracking system,
dashboard, PDF generator, and synthetic intelligence core.

## Features
- AI Agents (Synthetic, Strategic, Deep)
- Automatic CV/Resume Generator (PDF)
- Telegram Notifications
- Auto Apply to Jobs
- Employer Dashboard
- Full Authentication
- Secure Backend (JWT)
- Self-healing security AI
- Responsive front-end
- Dockerized deployment
- Nginx reverse-proxy

## Tech Stack
- FastAPI backend
- SQLite/PostgreSQL
- HTML5/CSS3/JS frontend
- AI agent system (Python)
- Docker + Nginx
- Fail2Ban + Firewall rules
